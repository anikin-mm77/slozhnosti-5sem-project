В данном репозитории находится проект с решением задачи о покраске графа в наименьшее количество цветов. 

Что из себя представляет проек:
  1. Файл "Услвоие задачи" - в нем ставится задача о покраске графа на математическом языке.
  2. Файл "solution.py" - в нем имплиментирована основная функция, которая раскрашивает граф
  3. Директория "data" - в ней содержатся файлы для тестирования алгоритма
  4. Файл "test.py" - в нем запускается тестирование реализованного алгоритма и формируется отчет об этом тестировании
