В данном репозитории находится проект с решением задачи о покраске графа в наименьшее количество цветов. 

Что из себя представляет проект:
  1. Файл description.pdf - описание проекта (отчет)
  2. Файл "Услвоие задачи" - в нем ставится задача о покраске графа на математическом языке
  4. Директория "data" - в ней содержатся файлы для тестирования алгоритма
  5. Файл "requirements.txt" - в нем содержатся необходимые зависимости
  6. Файл "solution.py" - в нем имплиментирована основная функция, которая раскрашивает граф
  7. Файл "test.py" - в нем запускается тестирование реализованного алгоритма и формируется отчет об этом тестировании
  8. Файл "run.sh" - bash скрипт, который запустит код

Как воспроизвести решение:
  1. Запустить bash скрипт "run.sh"
     
Альтернатива, если что-то не работает:
  1. Открыть проект в любой IDE (VS code)
  2. Создать виртуальное окружение (может создаться по умолчанию)
  3. Выполнить команду "pip3 install requirements.txt" и запустить файл test.py

Образ результата:

После запуска программа создаст файл report.txt - текстовый файл, в котором будет отчет о тестировании (все ли корректно, все ли критерии пройдены)
